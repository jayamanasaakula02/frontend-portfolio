# Frontend Developer Portfolio

A modern, professional portfolio website built with React JS, featuring a dark/light mode toggle, smooth transitions, and responsive design.

## Features
- **Modern UI/UX**: Clean design with vibrant accents and glassmorphism.
- **Theme Support**: Integrated light and dark mode with persistence.
- **Smooth Animations**: Powered by Framer Motion for page transitions and element reveals.
- **Dynamic Projects**: Dedicated detail pages for each project using React Router.
- **Responsive**: Fully optimized for Mobile, Tablet, and Desktop.
- **Interactive**: Hover effects, animated skill bars, and interactive timeline.

## Tech Stack
- **Frontend**: React (Functional Components, Hooks)
- **Styling**: CSS Modules, CSS Variables
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM (v6)

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Folder Structure
- `src/components`: Reusable UI elements like Navbar, Footer, and ProjectCards.
- `src/pages`: Main page components (Home, About, Skills, Projects, etc.).
- `src/context`: State management for themes.
- `src/styles`: Global variables and base styling.

## Customization
- Edit `src/styles/variables.css` to change the color palette.
- Update project data in `src/pages/Projects/Projects.jsx` and `src/pages/ProjectDetail/ProjectDetail.jsx`.
- Replace placeholders with your own info in the respective page components.

Enjoy building your professional brand!
